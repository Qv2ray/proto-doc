Config

Config is the master config of V2Ray. V2Ray takes this config as input and

functions accordingly.

FieldTypeLabelDescription
inbound InboundHandlerConfig repeated

Inbound handler configurations. Must have at least one item.

outbound OutboundHandlerConfig repeated

Outbound handler configurations. Must have at least one item. The first item is used as default for routing.

app common.serial.TypedMessage repeated

App is for configurations of all features in V2Ray. A feature must implement the Feature interface, and its config type must be registered through common.RegisterConfig.

transport transport.Config

Deprecated. Transport settings. Deprecated. Each inbound and outbound should choose their own transport config. Date to remove: 2020-01-13

extension common.serial.TypedMessage repeated

Configuration for extensions. The config may not work if corresponding extension is not loaded into V2Ray. V2Ray will ignore such config during initialization.

InboundHandlerConfig

InboundHandlerConfig is the configuration for inbound handler.

FieldTypeLabelDescription
tag string

Tag of the inbound handler. The tag must be unique among all inbound handlers

receiver_settings common.serial.TypedMessage

Settings for how this inbound proxy is handled.

proxy_settings common.serial.TypedMessage

Settings for inbound proxy. Must be one of the inbound proxies.

OutboundHandlerConfig

OutboundHandlerConfig is the configuration for outbound handler.

FieldTypeLabelDescription
tag string

Tag of this outbound handler.

sender_settings common.serial.TypedMessage

Settings for how to dial connection for this outbound handler.

proxy_settings common.serial.TypedMessage

Settings for this outbound proxy. Must be one of the outbound proxies.

expire int64

If not zero, this outbound will be expired in seconds. Not used for now.

comment string

Comment of this outbound handler. Not used for now.
